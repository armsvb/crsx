Joel stands for Java Object Expression Language.

Syntax

E ::= () | ( E1 ;...; Elast )
    | New[class, E1,...] | Invoke[Eobject, method, E1,...] | Static[class, method, argument,...]
    | NewArray[class, Elength] | ArraySet[Earray, Eindex, Evalue] | ArrayGet[E, index]
    | Null | False | True | Integer[term] | String[term] | Character[term]
    | Let[type, Evalue, variable . Ebody] | GenerateSymbol[prefix]
    | EnvironmentSet[name, Evalue, Ebody] | EnvironmentGet[name] .

Semantics

Expressions correspond to Java object operations in a fairly one to one way:
()
Do nothing; return void.

( E1 ;...; Elast )
Evaluate the expressions in sequence and return the result of evaluating the last. The expression stack from one E is passed to the next.

New[class, E1,...]
Return new instance of class using constructor taking (result of evaluating) the arguments E1,...

Invoke[Eobject, method, E1,...]
Return result of invoking the method of (evaluated) Eobject on (result of evaluating) the arguments E1,...

Static[class, method, argument,...]
Return result of invoking the static method of class on (result of evaluating) the arguments E1,...

NewArray[class, Elength]
Return new array instance with (result of evaluating) Elength members of class (initialized to null).

ArraySet[Earray, Eindex, Evalue]
Evaluate Earray and then set the Eindex'th member to Evalue; return void.

ArrayGet[Earray, Eindex]
Return result of evaluating Earray and then getting the Eindex'th member.

Null
Java null constant.

False
Java false constant.

True
Java true constant.

Integer[integer]
Java integer constant.

String[string]
Java string constant.

Character[character]
Java character constant.

Let[type, Evalue, variable . Ebody]
Evaluate Evalue and bind it to variable of type before evaluating Ebody.

GenerateSymbol[prefix]
Return string that starts with prefix and is different from all other such generated symbols.

EnvironmentSet[name, Evalue]
Set the environment variable name to the (result of evaluating) Evalue; return void.

EnvironmentGet[name, Edefault]
Return value of name environment variable or Edefault value if unset.
